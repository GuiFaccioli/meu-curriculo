# Currículo Online — Guilherme Faccioli

## Deploy

Deploy publicado na Vercel:

https://guilherme-faccioli-dev.vercel.app/

Currículo online desenvolvido em React + TypeScript para apresentar perfil profissional,
experiências, projetos, formação, certificações e competências técnicas de forma clara,
responsiva e com CTAs mais rastreáveis.

## Design e interações

O portfólio foi atualizado com uma linguagem editorial mais limpa, botões claros para GitHub,
LinkedIn, currículo e contato, além de cards de projeto com links e atributos prontos para
tracking futuro.

Detalhes da implementação:

[`docs/design-implementation.md`](docs/design-implementation.md)

## Tracking

Este projeto está preparado para receber Google Tag Manager via variável de ambiente.
Veja a documentação completa em:

[`docs/tracking/gtm-ga4.md`](docs/tracking/gtm-ga4.md)
