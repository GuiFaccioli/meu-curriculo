# Currículo Online — Guilherme Faccioli

## Deploy

Deploy publicado na Vercel:

https://guilherme-faccioli-dev.vercel.app/


Currículo online desenvolvido em React + TypeScript para apresentar perfil profissional, experiências, projetos, formação, certificações e competências técnicas de forma clara e responsiva.

## Tracking

Este projeto está preparado para receber Google Tag Manager via variável de ambiente.
Veja a documentação completa em:

[`docs/tracking/gtm-ga4.md`](docs/tracking/gtm-ga4.md)

## Tecnologias

- React
- TypeScript
- Vite
- CSS

## Como rodar localmente

```bash
npm install
npm run dev
```

## Como gerar build

```bash
npm run build
```

## Deploy

Deploy publicado na Netlify:

https://guilherme-faccioli-dev.netlify.app/

## Configuração Netlify

Build command:

```bash
npm run build
```

Publish directory:

```bash
dist
```
